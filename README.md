AUTOFLOWUSA — Official Pre‑Launch Landing Page


AUTOFLOWUSA is the official web presence for the AUTOFLOW hardware system — a next‑generation automotive automation device designed for reliability, simplicity, and real‑world performance. This repository contains the static website deployed through Cloudflare Pages and maintained via GitHub for continuous, version‑controlled updates.


───


🔍 Project Overview

The AUTOFLOWUSA website serves as the central hub for:


• Early‑access email capture

• Product awareness and pre‑launch marketing

• Feature previews and brand positioning

• Rapid iteration during development

• SEO‑optimized visibility for the AUTOFLOW brand


The site is intentionally lightweight, fast, and mobile‑optimized to support high conversion rates and clean deployment cycles.


───


🚀 Tech Stack

• HTML5 / CSS3 — Framework‑free, fast, and portable

• Cloudflare Pages — Global CDN, instant cache invalidation, automatic deployments

• GitHub — Version control, collaboration, and CI/CD integration

• Formspree (optional) — Email capture endpoint for early adopters

⚡ Deployment Workflow

AUTOFLOWUSA uses a GitHub → Cloudflare Pages pipeline:


1. Commit changes to the main branch

2. Cloudflare automatically pulls the update

3. Build completes in seconds

4. Site deploys globally to autoflowusa.com


No manual uploads. No downtime. No rebuild scripts.
